先给第一次听说这个价格的人解释一下。

## 搬瓦工各价位套餐速度对比

想选对套餐，就要搞清楚搬瓦工几条主要线路的区别。速度的核心决定因素是**线路类型**，而不是单纯看配置数字。

**搬瓦工线路等级从低到高大致是：**

普通线路（HE / 163 骨干网）→ CN2 GT → CN2 GIA → 香港 / 东京 CN2 GIA

普通线路晚高峰容易拥堵，速度有明显波动；CN2 GT 稍好，但电信方向已经不全程走 CN2；CN2 GIA 是目前主流用户的首选，三网回程走中国电信 CN2 骨干，延迟和丢包控制得很扎实；香港/东京 CN2 GIA 距离近，广东方向的用户延迟能压到非常低，但价格也最高。

实际用下来，CN2 GIA-E 套餐（洛杉矶 DC6/DC9）的表现是这样的：电信方向延迟大概在 130ms 上下，三网丢包接近 0，晚高峰基本不掉速，跑满宽带没有明显上限感。
